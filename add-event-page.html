Add new event to your calendar

How would you like to call your event?

You can add short description, where other participants will be able to find out more about the event, but it is not necessary. As for example, you can write there event venue.

Let's set the start date of the event.

Format: "YYYY-MM-DD".

Let's set the end date of the event.

Format: "YYYY-MM-DD".

You can also add the start time, but it is not necessary.

Format: "HH:MM".

And you add the end time, but it is not necessary too.

Format: "HH:MM".

For example, if you set:

Title: Summer Camp
About: Camp Harbor View,Plympton St. 46, Boston, Massachusetts.
Start date: 2025-06-02
End date: 2025-06-09
Start time: 10:00
End time: 20:00

It will mean that 'Summer Camp' will last from 10 a.m. 02/03/25 to 8 p.m. 09/06/25.

You can add participants to the event on Calendar -> More details (of the particular event) -> Participants.
